About

This is an initial attempt at rebuilding the Nemastocks database! I am using Flask, Flask-SQLAlchemy, and Flask-Bootstrap DJANGO. It has been a fun project so far, and I hope to continue to improve it over time.

It is hosted by UCSC at http://arriberelab.pbsci.ucsc.edu/ but hopefully you know that since you're here...

There are a few remaining major features that need to be implemented (listed below). Currently, all(?) previous features of the original Nemastocks database have been implemented.
